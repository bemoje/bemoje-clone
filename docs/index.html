clone

Deep and shallow clone. Merged forks of https://github.com/jonschlinkert/shallow-clone and https://github.com/jonschlinkert/clone-deep.

clone(value: any, deep: boolean, instanceClone: boolean)
Parameters
value (any) The value to clone
deep (boolean = false) Whether or not to do a deep clone
instanceClone (boolean = true) Whether or not to deep clone custom objects.
